DATASET ORIGINAL

Para este trabajo se utilizó el dataset Fruits-360, que contiene imágenes de frutas en formato JPG
organizadas en carpetas por clase (Training y Test).

Debido al tamaño del dataset completo, no se ha subido al repositorio de GitHub, ya que supera el límite de 100 MB por archivo. El dataset puede descargarse desde la siguiente fuente oficial:

- Nombre: Fruits-360
- Fuente: Kaggle
- Estructura original: LICENSE, README.md, Training/, Test/

En el código del proyecto (carpeta code/), se asume esta estructura original para cargar y preparar los datos.
